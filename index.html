Pertanyaan Umum Seputar Rebornian48

Karena dengan menggunakan fasilitas hosting yang disediakan oleh github, maka sampai sekarang admin sama sekali tidak membayar biaya hosting.

Karena sekarang diperuntukkannya adalah situs web uji coba dan rencananya masih menggunakan fasilitas dari Github, tapi tidak memungkiri jika ke depannya bisa dimigrasikan ke situs web yang berbayar.

Alasan admin menggunakan fasilitas github, selain hosting tidak membayar sepeserpun (sampai sekarang), kode yang diunggah di situs github dapat diambil seluruhnya dan dimodifikasi sebagian maupun seluruhnya.

Karena pada dasarnya sederhana saja, admin hanya merupakan fans JKT48, bukan bagian dari member atau pun staff JKT48. Hal itu lah yang membuat admin menuliskan Unofficial karena memang data yang ada di sini merupakan data tidak resmi, meskipun yang diambil dan diolah berasal dari situs web JKT48. Namun ada juga yang diambil dari Wikipedia baik yang bahasa Inggris dan bahasa Indonesia, serta beberapa situs pendukung lainnya.
  • Situs Web Official JKT48
  • Wikipedia Bahasa Indonesia JKT48
  • Wikipedia Bahasa Inggris JKT48
  • Situs Web AKB48 Fandom
  • Situs Web Wiki Stage 48

Ke depan, rencananya akan dibuat pedoman pembuatan website sorter agar dapat dikembangkan atau pun dimodifikasi oleh orang lain.
